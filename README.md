# 🏠 House Price Prediction Dashboard

## 📌 Project Overview

This project builds an **end-to-end Machine Learning pipeline** to predict house prices using the Kaggle **House Prices dataset**.
The project includes **data analysis, preprocessing, model training, and a Streamlit web dashboard** that allows users to enter house features and get predicted prices.

The goal of this project is to demonstrate practical skills in **data science, machine learning, and model deployment**.

---

## 🚀 Features

* Exploratory Data Analysis (EDA)
* Handling missing values
* Encoding categorical variables
* Feature selection
* Model training and evaluation
* Random Forest regression model
* Interactive **Streamlit dashboard**
* User login system
* Real-time house price prediction

---

## 📊 Dataset

Dataset source: Kaggle

The dataset contains housing features such as:

* Overall Quality
* Living Area (GrLivArea)
* Garage Size
* Basement Area
* Number of Bathrooms
* Year Built
* Lot Area

Target variable:

* **SalePrice**

---

## 🧠 Machine Learning Workflow

### 1️⃣ Data Preprocessing

* Handle missing values
* Encode categorical variables using **One-Hot Encoding**
* Feature engineering
* Data scaling and cleaning

### 2️⃣ Model Training

Algorithms tested:

* Linear Regression
* Random Forest Regressor

Final model used:
**Random Forest Regressor**

### 3️⃣ Model Evaluation

Evaluation metrics used:

* R² Score
* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

## 💻 Streamlit Dashboard

The project includes an interactive dashboard where users can:

* Input house features
* Predict house price instantly
* View dataset insights
* Visualize feature relationships

Run the dashboard:

```bash
streamlit run house_price_app.py
```
